# 🧮 C# Calculator Application

A simple and user-friendly **Calculator Application** built using **C# and .NET (Windows Forms)**.
This application performs basic arithmetic operations with a clean graphical interface.

---

## 📌 About The Project

This project is a desktop-based calculator developed using **C#**.
It demonstrates core concepts of:

* 🧠 Object-Oriented Programming (OOP)
* 🖥️ Windows Forms UI Design
* ⚙️ Event Handling in C#

💡 This calculator is designed for performing daily calculations efficiently.

---

## ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧹 Clear / Reset button
* 🧮 Real-time input and output display
* 🖱️ Button-based user interaction

---

## 🛠️ Technologies Used

* 💻 C#
* 🖥️ .NET Framework / .NET Windows Forms
* 🧰 Visual Studio
